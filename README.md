# pipecraft
Minecraft from scratch using python 

## Requirements

- Python 3.8
- PyOpenGL
- Pygame

Install the requirements with the command below:
```shell
$ pip install PyOpenGL Pygame
```

## Play

You just need to run the following:
```python
python main.py
```
